## Paranoid  偏执狂

高度洞察力与极度悲观的结合使得他总是做最坏的打算！

+ +可能会识别出新招募成员的特性
+ +总是留心危险
+ -可能脾气很坏

## Oblivious  不在乎

考虑到拒绝他人带来的神奇功绩，这是一种不是很敏感的豁达的人生观！

+ +可能避免陷入绝望
+ -洞察力和智力会得到差劲的骰子点数

## Charming 迷人

在别人看来，有着良好的态度和完全的魅力，这种人很会沟通！

+ +利用魅力获取免费事物或避开不利情形
+ -不能使用魅力规避一切

## Irritating  烦人  

这种人认为他们非非非非常酷。但是并没有！他们只是很烦人！

+ +劝告他人“别激动”
+ -激怒几乎每一个人

## Bandit  强盗

在后末日时代靠不住的特点也可能成为优势。

+ +偷抢别人的东西
+ -偷抢自己人的东西
+ -定时炸弹

## Mysterious Past  谜之过去

这种人...有着谜一样的过往......

+ +战斗技能随机加3点
+ -随机和极端的个性！

## Practical  实事求是

有着种种使用的技能，但有些无趣。

+ +战斗技能随机加3点
+ +辅助技能随机加3点
+ -无趣的性格

## Warrior  斗士

有着扎实的重击僵尸的本领，但性格平庸。

+ +力量加2点
+ +体力加2点
+ -无趣的性格

## Fierce Tempered  暴脾气

某人脾气不好！这种攻击性情绪可能会派上用场。

+ +力量额外加1点
+ -冷静程度低带来坏脾气
+ +脾气有时能带来好处

## Calm Under Fire  临危不乱

镇定自若。在射击一群僵尸时会很有用！

+ +冷静程度高使得非常冷静
+ 射击额外加1点

## Nurturing  仁爱

总为他人着想！不知为何还没成为僵尸的食物。

+ +医疗多2点！
+ +态度极好！
+ -也许在装模做样

## Inventive  善于创新

喜欢机械，比如汽车。

+ +修理多2点！
+ +智力检查极好！
+ -也许需要更多的练习才能修车

## Frantic Whiner  疯狂啜泣者

算是那种脆弱的人。但相当擅长逃跑！

+ +跑动速度加成！
+ -比常人少打一下
+ -态度很差

## Resilient Type  坚韧类型

非常坚强！有肌肉！不会做长期规划。

+ +比常人多扛一下伤害！
+ -智力得到差劲的骰子点数

## BERSERK!  狂战士！

毁掉所有僵尸！绝不后退！哦，不好，我被咬了！

+ +力量，体力，射击多4点！！
+ +能超出上述上限1点！！！
+ 超级低的性格状态！被咬一下就死！！

## Civilized  讲文明

保持着世界走向荒唐之前的价值观。不好也不坏。

+ 超高的忠诚和性格状态！
+ 医疗和修理技能多3点！！
+ -永久且严重的士气惩罚！！

## Sound Sleeper  酣睡者

无时无处不在睡觉。睡得最香！

+ +超人般的睡眠！只需很少的睡眠时间。
+ +永远不会有疲劳状态，永远。

## Gourmand  贪吃者

非常感激这些旧罐头！

+ +饱餐后恢复1点士气，直到满为止！生命值上限额外加1点！
+ -多吃一点，吃3份食物而非2份。食物短缺士气会骤降！

## City  Seeker  城市探索者

被祝福以有趣的人生！

+ +在城市搜刮事件中有更高几率找到奇特的建筑或人物！一般几率：10%  带特性：50%
+ +双倍几率找到稀有的交易营地！

## Travel Light  轻装上阵

携带得更少所以你能跑得更快一点！

+ +跑动速度加成！
+ -每次只能带两样武器！

## Hero Type  英雄类型

我将永远是你的保镖。呃，就一回。

+ +在文字事件中拯救一名队友免于死亡！
+ -每局游戏仅救一次！
+ +开局自带最高忠诚！

## Tiny Eater  节食者

比常人少吃很多！吃一两口！

+ +吃饱只需1份食物！！
+ -力量上限受降2点的惩罚！
+ -体力上限受降1点的惩罚。

## Anime Fan 日漫迷

好像是我看的僵尸动漫一样！

+ 武器替代为强力的武士刀。
+ -即使死掉也不能丢下武士刀。
+ -警告：防止士气满格！小心日漫！

## Specialist  专家

有点过于专注一种打击僵尸的方法。

+ 力量，体力，射击多2点
+ +能超出上述上限1点！！
+ -每次只能带一样武器！！

## Grappler  格斗者

如果僵尸倒地了，你能抬起他们！

+扔出或猛摔举起来的僵尸！

-无趣的性格

## Phoenix  不死鸟

凤凰涅槃！\*当成翅膀一样振动手臂\*

+ +在死亡中满血复活！！！
+ -第一次死亡后丢失此特性。

## Fireproof  浴火人  

不可燃？不可烧？让我们就叫他浴火人吧。

+ +无法燃烧！站立在火的中央吧！
