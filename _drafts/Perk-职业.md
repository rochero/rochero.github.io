注：以下为满星状态下的描述

## Mechanic  技工

有车就有活路！注：不一定是个**称职**的技工。

+ +初始携带扳手
+ +修理加2点！
+ +体力加2点！

## Athlete  运动员 

运动员的出身能很好地转变成生存能力。

+ +初始携带随机的运动器材武器
+ +体力加2点！
+ +力量加2点！

## Fighter  战士  

这个人一直是个真正的战士。

+ +初始携带短柄斧头
+ +力量加2点！
+ +射击加2点！

## Surgeon  外科医生

对解剖的直觉很准，手很稳。

+ 初始携带手术刀
+ +医疗加2点！
+ +射击加2点！

## Gungineer  枪械师

擅长修理枪械及各种东西。

+ +初始携带撬棍
+ +射击加2点！
+ +修理加2点！

## Martial Artist  武术家

没武器？没问题！

+ +出场自带强力二连击的徒手攻击！！
+ -拒绝使用枪炮，链锯等武器！！
+ +体力加3点，力量加1点！
+ +体力可以超出通常上限1点！

## Big Bruiser  大拳击手

行走缓慢，但击打很猛烈，能给予重创！

+ +出场自带更好的徒手攻击！比常人多扛一下伤害！
+ -走路速度更慢！
+ +力量加3点！

## Gun Collector  枪械收集者

你知道贮藏所有那些枪械往往很有用！在最后的时候。你知道的！

+ +射击加3点！
+ +初始携带一把霰弹枪，步枪，手枪，以及子弹！！

## Health Care  保健

傻瓜总会被僵尸咬到。最好做准备！

+ +初始携带7份医疗补给！
+ +医疗加2点！

## Car Nut  汽车迷

其实当下做个热衷于汽车的人是超级强大的！

+ +初始携带更好的车
+ +修理加3点！

## Megabuff  十分健硕

可能健硕到无法控制。能轻易抬起并扔出沉重的家具！

+ +初始携带金属管
+ +力量加4点！

## Ultrafit  极其健康

瑜伽训练真的有回报！

+ 初始携带燕麦棒
+ +体力加4点！

## Friend of Dog  狗的朋友

这个人很爱狗，而且往往会碰到狗！

+ +愤怒的狗可能会变得友好
+ +在随机招募中有70%的额外几率会是狗

## Explorer  探索者

一些人在开车时恰有非常好的方向感！

+ +在`永远在搜刮`事件中获得额外地点的选择
+ +`永远在搜刮`事件再刷新后消耗更少的汽油

## Ex-Wrestler  前摔跤手

在悲剧性灾难来临之前也曾是竞技场上的王者！

+ +抬起倒地的僵尸，然后扔出去或猛摔他们！
+ +体力加2点
+ +力量加1点

## Hidden Potential  隐藏的潜质

看似谦逊，但训练后超过常人

+ +体力或力量随机加1点并且可以超出上限1点！
+ +未显露的技能属性加1点
+ +体力或力量最后随机加1点

## Top Seller  顶级推销员

好的推销技巧依然有适用之处！

+ +当售卖武器给购买者时获得额外的1-5份食物，数量取决于初始售价
+ +当用食物充当强盗通行费时免受士气惩罚！
+ +强盗的通行费会变低！

## Pathfinder  探路人

远足的经历有助于避免林区恐怖活动！

+ +移除每次弃车后的徒步第一天的惩罚事件
+ +有75%几率跳过徒步第二天的惩罚事件

## Trademark Weapon  专有武器

以持有特殊标识的武器而在人群中显眼！

+ +开局默认携带铝制球棒。初始武器可能会随人物特性而改变！
+ -即使死掉也不能丢下初始武器
+ +体力加1点！
+ +射击加1点！

## Natural Shot  天生枪手

天生的射击好手！只是需要更多的训练和大概一把枪以及子弹！

+ +射击加2点！
+ +射击可以超出通常上限2点！

## T\*S\*T\*C  强壮到无法控制

确实太健硕了！**拒绝**做健心运动！

+ +开局满力量！
+ +可能可以训练力量超出上限6次！

## Shield of Hope  希望守护者

保持十足的意志力以经受磨难！

+ +如果士气低于 :( 则免受一次致命打击
+ -依然会在即死事件中死去
+ -每次躲过一劫后士气降2点

## Bow and Arrows  弓箭手

弓箭术最终成了实用的爱好！

+ +初始携带无声强弓，并附带最多30支箭
+ -即使死掉也不能丢下弓
+ +在两次行动之间能制作箭，最多10支！
