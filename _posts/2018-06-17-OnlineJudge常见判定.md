---
title: OnlineJudge常见判定
description: 刷题到最后都是需要英文基础的，看的懂错误是什才能对症下药
category: 基础 学习
---

| 中文       | 英文                       | 可能原因                                 |
| ---------- | -------------------------- | ---------------------------------------- |
| 答案正确   | Accepted(AC)               |                                          |
| 答案错误   | Wrong Answer(WA)           | 忽略了细节，数值精度损失或溢出，审题不清 |
| 格式错误   | Presentation Error(PE)     | 空格换行符不符合要求                     |
| 编译错误   | Compilation Error(CE)      |                                          |
| 运行时错误 | Runtime Error(RE)          | 段错误，浮点错误                         |
| 段错误     | Segmentation Fault(SF)     | 数组越界，堆栈溢出                       |
| 浮点错误   | Floating Point Error(FPE)  | 运算除0                                  |
| 运行超时   | Time Limit Exceed(TLE)     | 算法效率低，死循环                       |
| 内存超限   | Memory Limit Exceeded(MLE) | 数组太大，死循环                         |
| 输出超限   | Output Limit Exceeded(OLE) | 死循环                                   |
| 函数受限   | Restricted Function(RF)    | 使用了危险的函数                         |
