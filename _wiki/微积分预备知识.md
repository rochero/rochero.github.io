#### 特殊分段函数实例

- 符号函数 sign function

$$
y = sgnx=\begin{cases} x&=&1 & x > 0 \\ x &= & 0 & x  = 0 \\ x &=& -1 & x < 0\end{cases}
$$



- 绝对值函数 absolute value function

$$
y=\mid x\mid = \begin{cases}x & x \geqslant 0 \\ -x & x < 0 \end{cases}
$$



- 取整函数 intergral function

$$
y = [x](不超过x的最大整数部分)
$$

- 狄利克雷函数 dirichlet function

$$
y = \begin{cases} 1 & x是有理数 \\ 0 & x是无理数\end{cases}
$$

#### 初等函数 elementary function

- 基本初等函数 basic elementary function

  - 对数函数 logarithmic function
  - 三角函数 trigonometric function
  - 反三角函数 inverse trigonometric function

- 注：
  - 符号函数不是初等函数。初等函数是由基本初等函数构成的，一定是连续的。
  - 绝对值函数是初等函数。分段函数也可能能用一个式子表示，可能是初等函数。
